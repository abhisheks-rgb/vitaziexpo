# VitaziApp (Expo)

## Run in iOS Simulator — 3 steps

```bash
# 1. Install dependencies
npm install

# 2. Start Expo
npx expo start

# 3. Press 'i' in the terminal to open iOS simulator
```

That's it! No Xcode build needed, no pod install, no Metro setup.

## Notes
- Press 'i' for iOS simulator
- Press 'a' for Android
- Shake simulator (Cmd+Ctrl+Z) to reload
- Light/dark mode: Simulator > Device > Appearance
